Canvas笔记

2022-02-15

基本用法

html 写法：

<canvas id="canvas" width="150" height="150"></canvas>

渲染上下文，可以用来绘制想要的内容。传统意义上说的 canvas 的渲染上下文叫做 2d，而 WebGL 的渲染上下文叫做 webgl

let canvas = document.getElementById('canvas');
let ctx = canvas.getContext('2d');

绘制图形

坐标系统

左上角为原点，x 轴向右，y 轴向下。

绘制矩形

canvas 只支持两种形式的图形绘制：矩形和路径。

矩形的绘制有三种方法：

fillRect(x, y, width, height) // 绘制一个填充的矩形
strokeRect(x, y, width, height) // 绘制一个矩形的边框
clearRect(x, y, width, height) // 指定清除矩形的区域

路径绘制

步骤：

1、创建路径起始点

2、使用画图命令去画出路径

3、封闭路径

4、路径生成后，即可用描边或填充路径区域来渲染图形

路径相关的函数：

beginPath() // 新建一条路径，生成之后，图形绘制命令被指向到路径上生成路径。
closePath() // 闭合路径之后图形绘制命令又重新指向到上下文中。
// 并不是必须的，如果最后一个语句回到了起始点，则什么都不做。
stroke() // 通过线条来绘制图形轮廓。
fill() // 通过填充路径的内容区域生成实心的图形。

画图命令：

// 移动位置
moveTo(x, y) // 将绘制的起始点移动到某个位置

// 画线
lineTo(x, y) // 从起始点画出一条线到另一个点，并把起始点移动到这个点。

// 画弧线
// 画一个以 x, y 为圆心的弧。其他参数都在函数里。
// startAngle 的 0 是x 轴正方向。两个角度使用的是弧度而不是角度
arc(x, y, radius, startAngle, endAngle, anticlockwise)
// 根据给定的控制点和半径画一段圆弧，再以直线连接两个控制点。
arcTo(x1, y1, x2, y2, radius)

// 贝塞尔曲线
quadraticCurveTo(cp1x, cp1y, x, y) // 二次贝塞尔曲线
bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y) // 三次贝塞尔曲线

// 矩形
rect(x, y, width, height)

例子（三角形）：

ctx.beginPath();
ctx.moveTo(75, 50);
ctx.lineTo(100, 75);
ctx.lineTo(100, 25);
ctx.fill();

Path2D 对象

可以用来缓存和记录绘图命令。

new Path2D();     // 空的Path对象
new Path2D(path); // 克隆Path对象
new Path2D(d);    // 从SVG建立Path对象
Path2d.addPath(path[, transform]) // 添加了一条路径到当前路径，可能添加了一个矩变换阵

例子：

var ctx = canvas.getContext('2d');

var rectangle = new Path2D();
rectangle.rect(10, 10, 50, 50);

var circle = new Path2D();
circle.moveTo(125, 35);
circle.arc(100, 35, 25, 0, 2 * Math.PI);

ctx.stroke(rectangle);
ctx.fill(circle);

// svg
var p = new Path2D("M10 10 h 80 v 80 h -80 Z");

样式

颜色

ctx.fillStyle = color // 填充颜色（包括rgba）
ctx.strokeStyle = color // 轮廓颜色（包括rgba）
ctx.globalAlpha = value // 透明度

线

ctx.lineWidth = value // 线粗
ctx.Cap = type // butt round squire
ctx.lineJoin = type // round bevel miter
ctx.miterLimit = value // 数字
ctx.getLineDash()
ctx.setLineDash(segments)
ctx.lineDashOffset = value

一些问题:

1、线宽的问题：

有时候宽度 1 在渲染的时候会有各自一边是 0.5，不满一像素，这时候就会用更加浅的颜色替代，因此宽度不是完美的 1。有时候需要注意一下这个问题。

2、butt round squire 的区别：

3、round bevel miter 的区别：

4、miterLimit 的问题：

使用 miter 的话，线段的外侧边缘会被延伸交汇于一点上。线段之间夹角比较大时，交点不会太远，但随着夹角变小，交点距离会呈指数级增大。因此需要一个限制

miterLimit 属性就是用来设定外延交点与连接点的最大距离，如果交点距离大于此值，连接效果会变成 bevel。 $$ \text{miterLimit} = \frac{\max(\text{miterLength})}{\text{lineWidth}} = \frac{1}{\sin(\min\frac{\theta}{2})} $$ 5、setLineDash 方法和 lineDashOffset 属性

setLineDash 方法接受一个数组，来指定线段与间隙的交替；lineDashOffset 属性设置起始偏移量

渐变对象

ctx.createLinearGredient(x1, y1, x2, y2) // 渐变的起点 x1, y1 到终点 x2,y2
ctx.createRadialGradient(x1, y1, r1, x2, y2, r2) // 前三个点定义 x1, y1 为原点，半径为 r1 的圆，后三个参数同理
gredient.addColorStop(position, color) // 添加颜色，position 代表 0-1 之间的数值，color 代表这个位置的颜色。addColorStop 方法可以添加多次，以在不同位置上画出不同的颜色。

可以将 gredient 对象赋值到 fillStyle, storkeStyle 上以实现渐变的效果。

图案对象

ctx.createPattern(image, type) // image 可以是 Image 对象，或另一个 canvas 对象。 type 必须是 repeat，repeat-x，repeat-y 和 no-repeat

同样，也是赋值给 fillStyle, storkeStyle 以实现图片效果。

阴影

ctx.shadowOffsetX = float
ctx.shadowOffsetY = float
ctx.shadowBlur = float
ctx.shadowColor = color

填充规则

ctx.fill('nonzero') // 或者 'evenodd'

绘制文本

ctx.fillText(text, x, y, [, maxWidth]) // 在 x, y 处填充指定的文本。绘制最大宽度可选。
ctx.strokeText(text, x, y, [, maxWidth]) // 绘制文本边框

文本样式

ctx.font = value
ctx.textAlign = value // start end left right center
ctx.textBaseline = value //  top hanging middle alphabetic ideographic bottom
ctx.direction = value // ltr rtl inherit

预测量文本宽度

ctx.measureText('...') // 测量 ... 文本的宽度、所在像素等属性，返回 TextMetrics 对象

图像

引入图像

引入图像的步骤：

1、获得一个指向 HTMLImageElement 的对象或者另一个 canvas 元素作为源。

2、使用 drawImage() 函数将图片绘制到画布上。

图像的来源可以有四种：

1、new Image() 构建的 HTMLImageElement 对象

2、从 <video> 中获取的视频帧 HTMLVideoElement

3、从另一个 <canvas> 获得的 HTMLCanvasElemnt。

4、ImageBitmap 对象。

从头开始构建一个图像：

let img = new Image();
img.onload = () => {
    // drawImage
}
img.src = 'test.png';
img.src = 'data:imge/gif;base64,....' // data: url 方法。

绘制图像

drawImage(image, x, y) // image 是图像对象，x, y 是其在 canvas 中的起始位置
drawImage(image, x, y, width, height) // 多出来的 width height 是将图像缩放至这个大小
drawImage(image, sx, sy, sWidth, sHeight, dx, dy, dWidth, dHeihgt) // 切片功能，详见下图

控制缩放行为

可以通过调整 imageSmoothingEnabled 属性选择是否在缩放图像时使用平滑算法。默认值为 true。

变形（画布）

状态保存和恢复

ctx.save()
ctx.resotre()

移动

ctx.translate(x, y) // 将原点移动到 x, y 的位置
ctx.rotate(angle) // 以原点为中心点顺时针旋转 angle 弧度的角
ctx.scale(x, y) // x, y 取值从 -1 到 1，代表缩放的程度。
ctx.transform(a ,b, c, d, e ,f) // 旋转矩阵
ctx.setTransform(a, b, c, d, e, f) // 将当前的变形矩阵重置为单位矩阵，然后用相同的参数调用 transform 方法。
ctx.resetTransform() // 重置当前变形为单位矩阵

旋转矩阵：

$$ \begin{bmatrix} a && c && e \newline b && d && f \newline 0 && 0 && 1 \end{bmatrix} $$

组合

globalCompositeOperation

ctx.globalCompositeOperation = type // 图形互相遮盖时的策略，详见下方

裁剪路径

ctx.clip() // 将当前正在构建的路径转换为当前的裁剪路径。即最后是 endPath 就是绘制，clip 就是裁剪。

像素操作

ImageData

我们可以直接通过 ImageData 对象操纵像素数据。

ImageData 对象的属性：

1、width 宽度

2、height 高度

3、data：Uint8ClampedArray类型的一维数组，包含着RGBA格式的整型数据，范围在0至255之间（包括255）。包含高度 × 宽度 × 4 bytes数据，索引值从 0 到 (高度×宽度×4)-1

创建 / 获取一个 ImageData 对象：

ctx.createImageData(width, height) // 所有像素被预设为透明黑。
ctx.createImageData(anotherImageData) // 创建一个与 anotherImageData 像素相同的对象，同样是全黑。

ctx.getImageData(left, top, width, height)

保存图片

canvas.toDataURL('image/png')
canvas.toDataURL('image/jpeg', quality) // 可以选择 quality
canvas.toBlob(callback, type, encoderOptions) // 创建一个 Blob 对象
