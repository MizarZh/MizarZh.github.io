Node在linux上的更新问题

2022-01-21

概述

由于 node 在 linux 的包（指的是使用 apt install 安装的包）长期不更新，而手动下载再改 PATH 并进行管理十分麻烦，因此有一个小技巧，可以使用 apt install 安装到新版本。

使用

nodesource 将继续维护各种 linux distributions 的 apt install 更新。

运行以下脚本（Ubuntu 环境下。其中 setup_17.x 中的数字 17 可以替换成其他版本如 16、14等）：

curl -fsSL https://deb.nodesource.com/setup_17.x | sudo -E bash - # 这一句是为了获取安装脚本，以替代原来老旧的 node 路径。
sudo apt-get install -y nodejs # 用更新的路径安装 nodejs

即可完成新版本的安装。

Debian 系统与 Ubuntu 系统的安装脚本有些不同，详见 GitHub - nodesource/distributions: NodeSource Node.js Binary Distributions
