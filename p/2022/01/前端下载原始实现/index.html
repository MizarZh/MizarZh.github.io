前端下载原始实现

2022-01-24

如果服务器有公开的路径

调用 window.location 跳到指定的路径。

如果使用二进制传输

服务端：

内部传二进制类型，并设置 HTTP 头：

Content-Type: application/octet-stream

前端：

首先要设置 XMLHTTPRequest 的 responseType 为 arraybuffer，然后创建 blob 对象，最后创建连接下载。

// 设置 respnseType
XMLHttpRequest.responseType = 'arraybuffer'
// jquery:
$.ajax({
    xhrFields: {
        responseType = 'arraybuffer'
    }
})
// 其他什么 axios 应该也直接有类似选项
// 创建 Blob：
let blob = new Blob([res.data], { type: 'application/octet-stream' });

// 创建链接并下载
let URL = window.URL || window.webkitURL; // 获取 URL
let objectUrl = URL.createObjectURL(blob); // 创建 ObjectURL
// 创建链接
let a = document.createElement('a');
a.href = objectUrl;
a.download = fileName; //下载后文件名
document.body.appendChild(a);
a.click(); //点击下载
document.body.removeChild(a); //下载完成移除元素

window.URL.revokeObjectURL(objectUrl); // 销毁 ObjectURL
