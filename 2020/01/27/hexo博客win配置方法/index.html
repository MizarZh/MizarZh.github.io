hexo win10配置方法，并使用github作为博客地址

0、准备：
node.js官网下载node并安装
安装git for windows，以后将使用git bash作为命令行操作
配置git账号信息：
git config –global user.name “Firstname Lastname”
git config –global user.email “your-email@youremail.com“
github新建repository的地址需明明为your-github-name.github.io

1、npm命令在国内较慢，需要使用cnpm(taobao mirror)
命令：
npm install -g cnpm –registry=https://registry.npm.taobao.org

2、使用cnpm安装hexo
命令：
cnpm install -g hexo-cli

3、新建一个文件夹，里面将存储blog所需文件
命令：
mkdir xxx
cd xxx

4、初始化hexo
命令：
hexo init

5、安装hexo部署至github的工具
命令：
cnpm install –save hexo-deployer-git
（若出现ERROR Deployer not found: git错误，则是这个工具未安装）

其中文件_config.yml是配置文件
部署到github上需要配置deploy部分
type: ‘git’
添加两行：
repo: path-to-github-blog
branch: master
可在theme中更换主题

本地临时服务器：
hexo s
(start)

新建文档：
hexo n “name”
(new)
文档存放在/source/_post/中

生成文档：
hexo g
(generate)

部署文档到github上：
hexo d
(deploy)
第一次配置可能需要输入github的账号密码
若是git bash内提示账号相关的错误，返回第零步进行账号的配置
